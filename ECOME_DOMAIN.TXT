### Integration Point 5: Cancellation & Downgrade Logic

**If FlowShop Pro user downgrades to FlowShop Basic:**

1. User receives warning: "Your free domain (mybrandstore.com) will no longer be included. It will convert to a paid domain at $14.99/year, billed on your next domain renewal date. You can also transfer your domain to another registrar for free."
2. If user confirms downgrade: plan drops to $5/month, domain stays active but flagged for paid renewal
3. 30 days before domain renewal, user gets email: "Your domain mybrandstore.com will renew at $14.99. [Renew] [Transfer Out] [Let Expire]"

**If FlowShop Pro user cancels e-commerce entirely:**

1. User receives warning about both store deactivation and domain status
2. Store is deactivated, data retained for 90 days
3. Domain remains registered (Cloudflare still manages it) â€” user gets option to: renew at $14.99/year as standalone, transfer to another registrar for free, or let it expire
4. Subdomain stops working immediately upon cancellation

**If FlowShop Basic user cancels:**

1. Store deactivated, data retained 90 days
2. Any purchased domains remain registered â€” renewal at marketplace pricing
3. Subdomain stops working immediately
